Welcome to MailSpy

MailSpy is a cybersecurity tool designed to help users identify spoofing and phishing attempts hidden in email headers. Whether you're a beginner or a security analyst, this project offers a simple way to learn about and defend against email-based threats.

🔍 Features:

  • ✅ Analyze raw email headers in one click
  • ✅ Detect SPF, DKIM, and DMARC validation results
  • ✅ Fast and clean web interface
  • ✅ Built using Flask and Python
